# OWOT type declarations

Comprehensive TypeScript declaration files for OWOT scripting projects. This repo provides type safety and autocomplete for IDEs, with all functions, variables and classes documented in JSDoc.

## Usage

Include the declaration files in your project (ideally, in a separate folder), and add the folder to the include path of your `tsconfig.json`. Everything is in the global scope, so you don't need to worry about importing.
